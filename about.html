About My Mini Store

My Mini Store is a mock e-commerce website created as a final project for a web development class. The goal of this site is to demonstrate:

  • Multi-page website structure with consistent navigation
  • Use of HTML, CSS, and JavaScript for interactive features
  • Dynamic product details and a simple shopping cart
  • Form handling for a mock checkout experience

Technologies Used

  • HTML5 for the structure of each page
  • CSS for layout, colors, and consistent styling
  • JavaScript for dark mode, product filtering, cart logic, and checkout form behavior
  • localStorage to store cart items in the browser
